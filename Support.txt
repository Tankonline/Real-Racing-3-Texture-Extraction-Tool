-------------------------------------------------
INSTALLATION PROBLEMS
-------------------------------------------------

Problem:
"compressonatorcli.exe not found"

Fix:
Make sure AMD Compressor CLI is located in:

Tools\AMD Compressor CLI\compressonatorcli-4.5.52-win64\


Problem:
"PVRTexToolCLI.exe not found"

Fix:
Place it inside:

Tools\PvrTexTool CLI\


Problem:
"python is not recognized"

Fix:
Install Python and enable "Add to PATH"
OR use full python path in .bat files.


-------------------------------------------------
EXTRACTION ERRORS
-------------------------------------------------

Problem:
"No valid zlib blocks found"

Cause:
File is already unpacked.

Fix:
Use original .z file only.


Problem:
"Decompression failed"

Cause:
Corrupted or wrong file.

Fix:
Re-extract from game.


-------------------------------------------------
CONVERSION ERRORS
-------------------------------------------------

Problem:
"Invalid format specified"

Cause:
Wrong compression flag.

Fix:
Use supported formats only:
ETC2 (Android)
BC3 (PC)


Problem:
"No output files created"

Cause:
No PNG files found.

Fix:
Check:

Run_tool\PNG_Universal\


-------------------------------------------------
PACKING ERRORS
-------------------------------------------------

Problem:
".z file not created"

Cause:
DDS file missing or Python error.

Fix:
Check:

output_dds\
Tools\Zlib\pack_zlib.py


Problem:
".z file crashes game"

Cause:
Wrong resolution or format.

Fix:
Keep original size and alpha.


-------------------------------------------------
IOS PLATFORM NOTES
-------------------------------------------------

Important:

iOS support is limited.

Real Racing 3 iOS uses:

- PVR / PVRTC
- Metal formats
- Device-specific compression

Currently:

✔ Android supported
✔ PC supported
⚠ iOS experimental / unsupported

iOS assets may require:

PVRTexTool advanced settings
Manual encoding
MacOS tools


-------------------------------------------------
PERFORMANCE ISSUES
-------------------------------------------------

Problem:
Conversion is slow

Fix:
Close other programs.
Use SSD storage.
Process fewer files.


Problem:
Tool freezes

Fix:
Wait 1–2 minutes.
Large textures take time.


-------------------------------------------------
FILE MANAGEMENT ISSUES
-------------------------------------------------

Problem:
Files overwrite each other

Fix:
Do not rename outputs.
Process one mod at a time.


Problem:
Missing files

Fix:
Restore from backup.


-------------------------------------------------
SAFETY & BACKUPS
-------------------------------------------------

ALWAYS:

✔ Back up original game files
✔ Keep unmodified copies
✔ Store mods separately
✔ Test offline first
