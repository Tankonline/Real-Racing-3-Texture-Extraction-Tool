Real Racing 3 Texture Tools
Directory Structure
=========================


Root
----

Directory.txt
README.txt
Support.txt


Main Folders
------------

Converted to .z assets
→ Final repacked game-ready .z files

extracted_dds
→ Extracted DDS textures from game assets

Original files
→ Backup of original RR3 files (recommended)

output_dds
→ Temporary DDS files created during conversion

output_z
→ Temporary ZLIB output files

png_edit
→ Workspace for editing PNG textures


Run_tool
--------

batch_extraction.bat
→ Batch extract multiple .z files

dds_to_png.bat
→ Convert DDS files to PNG format

export_universal.bat
→ Export PNG files to Android & PC formats

png_to_game_ready.bat
→ Convert PNG → DDS → ZLIB (.z)

run.bat
→ Main extraction tool

PNG_Universal
→ Folder for universal PNG source files


Tools
-----

AMD Compressor
→ Compressonator GUI

AMD Compressor CLI
→ Command-line texture compressor

PvrTexTool
→ PowerVR texture tools installer

PvrTexTool CLI
→ PowerVR command-line converter

Python
→ Python installer

Zlib
→ ZLIB compression scripts and tools

noesis
→ Texture viewing and conversion tool

Paint.NET
→ Image editing software


universal_assets
----------------

android
→ ETC2 compressed textures (Android)

ios
→ Reserved / Experimental iOS output

pc
→ BC3 compressed textures (PC)


----------------------------------
Folder Workflow Summary
----------------------------------

Original Game Files  → Original files
Extracted DDS        → extracted_dds
Converted PNG        → PNG_Universal / png_edit
Temporary DDS        → output_dds
Final Z Files         → Converted to .z assets
Universal Assets     → universal_assets
